## Technologies Used

-   Next.js
-   Chakra-UI
-   Eslint and Prettier

## Usage

-   `npm install` - to install
-   `npm run dev` - for development
-   `npm run build` - to build project
-   `npm run prettier` - to run Prettier
-   `npm run lint` - to run Eslint
